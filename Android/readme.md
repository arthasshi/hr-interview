### 工作时间安排
多伦多和中国有12小时时差（冬季13小时）我们的工作时间
- 每周 1，4 团队语音会议
  - 夏季: 北京时间早上9：00，相当于多伦多时间晚上9：00
  - 冬季: 北京时间早上9：00，相当于多伦多时间晚上8：00

- 极少出现加班的情况。（过去一年里大概有几次，紧急bug）
- 安静的在家钻研技术打磨产品。不搞那些有用没用的 work hard play hard

期待每周5x8小时工作是有效率的工作。

- 无故连续不能提交进度
